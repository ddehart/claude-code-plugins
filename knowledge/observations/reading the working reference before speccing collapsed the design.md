The design conversation "changed completely" only once the actual reference implementation was read —
revealing it was all prose, no validator, no lifecycle machinery, thriving on ~1,150 lines across a few
files. The prior inflated spec had been written from imagined requirements rather than the observed
artifact. The restart began from the lesson the earlier version paid for: read the working system
end-to-end before speccing its generalization, and let its real weight set the target.
