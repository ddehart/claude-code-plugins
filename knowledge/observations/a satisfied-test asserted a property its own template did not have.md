A delta's suppression cadence used a marker whose category name was free-form prose invented each run, and
scanned for it literally in full. Its own `satisfied-test` asserted "a fixed marker" — true only of the
prefix. The test would have passed the defective version, because it was written by the same author in the
same pass as the thing it verifies, and encoded the author's belief about the artifact rather than its
properties. Caught by independent review; the fix corrected both the template and the test.
