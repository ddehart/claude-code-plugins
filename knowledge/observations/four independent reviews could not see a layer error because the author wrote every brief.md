A feature was built one abstraction layer below the request — recommending notes for individual nouns when
the ask was for new schema types. It passed through a spec, four agent briefs, five independent code
reviews and a live application before the requester caught it in one sentence. Every reviewer was asked
whether the implementation matched the stated intent; none was positioned to ask whether the stated intent
matched the request. The config that defines the term was open and read repeatedly across two days.
