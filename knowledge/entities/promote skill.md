# promote skill

The [[knowledge-commons plugin]] cross-graph promotion skill. Derives a *new, self-contained* note into a
target graph (never moves the source), stamps a non-resolving `domain:` field naming the source graph,
associates the claim with an existing attractor / open question / new question, and requires explicit human
approval. As of the day's releases it commits-and-pushes the named files it writes — an uncommitted
promotion is invisible to every other machine and session. It is the commons's only inlet; see
[[processing is local to each domain and the commons is receive-only via promotion]].
