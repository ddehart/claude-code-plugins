# processing is local to each domain and the commons is receive-only via promotion

Every domain keeps its own graph and processes its own material at home; the commons never processes raw
material and never reaches into project trees. `/promote` is the commons's *only* inlet, and it is uniform
for all domains — a single mechanism that stamps each derived claim with a non-resolving `domain:` field
naming its source graph. An earlier design fed chronicles into the commons directly and branched promotion
on a personal/professional posture; both were dropped as premature — the posture split solves a
confidentiality problem the human-approval gate already handles, and direct feeding forced the commons to
know about project files it should never touch.
